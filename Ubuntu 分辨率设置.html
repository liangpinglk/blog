Ubuntu 分辨率设置

最近使用Ubuntu时遇到个问题，使用2k屏幕的时发现没有2560*1440分辨率，显示效果就很难受，于是查了下，发现可以使用xrandr命令设置分辨率，本文内容主要转载自Ubuntu添加2560x1440分辨率

  • 1.使用xrandr 查看当前显示器所有分辨率，其中virtual1 是显示器的名字 ``` bash ~ xrandr Screen 0: minimum 1 x 1, current 2560 x 1440, maximum 16384 x 16384 Virtual1 connected primary 2560x1440+0+0 (normal left inverted right x axis y axis) 0mm x 0mm 800x600 60.00 + 60.32

- 2.添加分辨率
``` bash
cvt 2560 1440
  • 3.接着输入
    xrandr --newmode "2560x1440_60.00"  312.25  2560 2752 3024 3488  1440 1443 1448 1493 -hsync +vsync
  • 4.添加该模式，此时就可以在display settings 中看到2560*1440 的分辨率了，选择即可
    xrandr --addmode Virtual1 "2560x1440_60.00"
  • 5.以上步骤完成后在Ubuntu重启后，设置的分辨率会消失，如果要使其长期有效，则将以下命令拷贝到/etc/profile 文件中即可
    cvt 2560 1440
xrandr --newmode "2560x1440_60.00"  312.25  2560 2752 3024 3488  1440 1443 1448 1493 -hsync +vsync
xrandr --addmode Virtual1 "2560x1440_60.00"
    1920 * 1080 分辨率
